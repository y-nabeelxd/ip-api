IP Info API

A lightweight, self-hosted API for retrieving IP geolocation information

API Endpoint

GET https://https://your.project.vercel.appvercel.app/api/ipinfo?key=YOUR_API_KEY

Response Format

{
  "ip": "8.8.8.8",
  "city": "Mountain View",
  "country_code": "US",
  "country_name": "United States",
  "region_code": "CA",
  "region": "California"
}

How to Use

Include your API key as a query parameter:

fetch('https://https://your.project.vercel.appvercel.app/api/ipinfo?key=YOUR_API_KEY')
  .then(response => response.json())
  .then(data => console.log(data));

Features

  • Lightweight and fast
  • No rate limits
  • Returns only essential information
  • Easy to integrate
  • Self-hosted solution

Response Fields

  • ip - The IP address of the client
  • city - Detected city name
  • country_code - Two-letter country code (ISO 3166-1 alpha-2)
  • country_name - Full country name
  • region_code - Region/state code
  • region - Full region/state name
